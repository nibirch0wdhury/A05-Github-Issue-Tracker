**1️⃣ What is the difference between var, let, and const?**
All three are used to declare variables in JavaScript. The main difference is how they handle scope and reassignment. var is the older way and it is function scoped, which can sometimes lead to unexpected behavior. let and const were introduced in ES6 and they are block scoped, which makes them safer to use. let allows the variable value to be changed later, while const does not allow reassignment after it is declared.

**2️⃣ What is the spread operator (...) ?**
The spread operator is used to expand elements of an array or properties of an object. It helps copy or combine arrays and objects easily. Instead of manually looping through values, the spread operator can quickly duplicate or merge data, making the code shorter and cleaner.

**3️⃣ What is the difference between map(), filter(), and forEach()?**
These are array methods used to work with elements inside an array. forEach() goes through each element and performs an action but does not return a new array. map() also loops through the array, but it creates and returns a new array with modified values. filter() returns a new array as well, but it only includes elements that match a certain condition.

**4️⃣ What is an arrow function?**
An arrow function is a shorter syntax for writing functions in JavaScript, introduced in ES6. It makes the code more concise and easier to read. Arrow functions also handle the this keyword differently compared to regular functions, which can help avoid some common issues in callbacks or object methods.

**5️⃣ What are template literals?**
Template literals are a way to create strings using backticks instead of single or double quotes. They allow variables and expressions to be inserted directly inside a string, which makes string formatting easier. They also support multi-line strings without needing special characters.
